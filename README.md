# Aplicação de Mapa para Dispositivos Zabbix

## Descrição
Aplicação Dockerizada para visualização geográfica de dispositivos monitorados pelo Zabbix no mapa do Brasil.

## Pré-requisitos
- Docker e Docker Compose
- Acesso à API do Zabbix
- Dispositivos com coordenadas (latitude/longitude) configuradas no inventário
